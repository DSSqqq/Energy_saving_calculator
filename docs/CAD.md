# 3D CAD: секции здания

Документация по геометрии, стыковке секций, расчёту метрик и отображению размеров на странице здания (`BuildingDetailsPage`).

## Компоненты

| Файл | Назначение |
|------|------------|
| `frontend/src/BuildingDetailsPage.tsx` | UI секций, форма создания/редактирования, стыковка, блок «Расчёт геометрии» |
| `frontend/src/measures/cad/BuildingCadViewer.tsx` | Canvas: Solid / Wireframe / 2D План, камера, отрисовка модели |
| `frontend/src/measures/cad/footprint.ts` | Контур секции (прямоугольник, трапеция по сторонам С/Ю/В/З), стыковка |
| `frontend/src/measures/cad/geometryMetrics.ts` | Площадь стен, объём, площадь контуров (исключая стыки) |
| `frontend/src/measures/cad/dimensionDraw.ts` | Размерные линии на крыше со стрелками |
| `frontend/src/App.css` | Стили `.cad-grid`, `.cad-sections-card`, `.geometry-metrics-*`, `.checkbox--primary` |

## Контур секции (`footprint.ts`)

Порядок построения:

1. Если заданы все четыре стороны (С/Ю/В/З) — **трапеция/четырёхугольник** по длинам сторон (не обязательно прямоугольник).
2. Иначе — обход сторон по списку `sides` с поворотом на 90°.
3. Иначе — прямоугольник `length × width` с учётом `offset_x`, `offset_y`.

Координаты (вид сверху): **+X — юг**, **−Z — север**.

## Стыковка секций

В форме секции: «Пристыковать к существующей секции».

| Режим | Когда | Поведение |
|-------|--------|-----------|
| **Ребро к ребру** | Противоположные стороны (Запад↔Восток, Север↔Юг) | Стена примыкает вплотную вдоль всей стороны |
| **Угол к углу** | Перпендикулярные стороны (например Юг→Восток) | Г-образная форма |

Параметры:

- **Позиция вдоль стены** — от начала / по центру / от конца.
- **Сдвиг вдоль стены (м)** — смещение размерной линии стыковки.

Функция: `computeJoinOffset()` в `footprint.ts`.  
Смещения `offset_x`, `offset_y` сохраняются в API (`BuildingSection`).

## Расчёт геометрии

Блок под CAD «Расчёт геометрии здания»:

- **Площадь наружных стен** — Σ (длина ребра × высота); стыки между секциями не учитываются.
- **Строительный объём** — Σ (площадь контура × `height_outer`), не по габаритному прямоугольнику.
- **Площадь контуров** — сумма площадей полигонов секций.
- **Исключено (стыки)** — площадь общих стен между секциями (информационно).

Функция: `computeBuildingGeometryMetrics()` в `geometryMetrics.ts`.  
Обновляется вместе с live-preview при редактировании формы (`cadSections`).

## Размерные линии на крыше

На поверхности крыши каждой секции (y = `height_outer`):

- выносные пунктирные линии от углов;
- размерная линия параллельна стороне (смещена к центру блока);
- **стрелки** на обоих концах (как на архитектурном чертеже);
- подпись в рамке: `Север · 12 м` и т.п.

Режимы: Solid, Wireframe, 2D План.  
Функции: `buildRoofDimensions()`, `drawRoofDimensionWithWitness()` в `dimensionDraw.ts`.

## Сохранение секции

- Редактирование: `PATCH /api/building-sections/{id}/`
- Валидация в форме (без блокировки HTML5): ошибки показываются у кнопки «Сохранить»
- При включённой стыковке обязателен выбор целевой секции
- После сохранения — тихое обновление списка без полноэкранного loader
